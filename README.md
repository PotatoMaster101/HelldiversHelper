# Helldivers Helper
A helper tool for playing Helldivers. Automatically type in any stratagem combinations.

**Warning**: using this tool removes the immersion and stress when playing on high difficulty maps. Use at your own risk.

## Building
Build with [`dotnet`](https://dotnet.microsoft.com/):
```
$ dotnet build -c Release
```

## Running
```
$ dotnet run -c Release --project src/HelldiversHelper.Wpf
```

## Usage
Use the `Add` button to add a new stratagem hotkey. In game press the hotkey to auto type the stratagem combo. CTRL will not be pressed - hold down CTRL first before pressing the hotkey.
